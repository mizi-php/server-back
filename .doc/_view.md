### View Prepare

**this.script** Script no arquivo script.js da view

    [#this.script]

**this.style** Estilo no arquivo style.css da view

    [#this.style]

**view** Chama uma view

    [#view:view]

**this.view** Chama uma view dentro da view original

    [#this.view:view]

**url** Atalho para a criação de URLs

    [#url:...]
